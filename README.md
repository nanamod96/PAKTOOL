## Installing
1. Install Termux 
```
pkg update && pkg upgrade
pkg install git
termux-setup-storage
```
4. Clone this repository
```
git clone https://github.com/nanamod96/PAKTOOL.git
```
5. Open PAKTOOL directory and run setup.
```
cd PAKTOOL
chmod +x setup
./setup
```
6. DONE
# Next time when entering the tool, type "pak" and the tool will automatically start.
